# Sync
Open folder `NOTEs` in `Visual Studio Code`.

Run this command to build on local host.
```terminal
npx quartz build --serve
```

If there is no error, run this command to sync over `haolamnm.github.io/notes`.
```terminal
npx quartz sync
```
